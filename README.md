## Installation

Install Dependencies:  
`npm install`

Development:  
`npm run dev`

Storybook:  
`npm run storybook`

Linting:  
`npm run lint`

NextJS Build:  
`npm run build`

AWS Serverless Build:  
`npm build:serverless`
